========================================================================
   ПРИЛОЖЕНИЕ : Обзор проекта Project3
========================================================================

Это приложение Project3 создано автоматически с помощью 
мастера приложений. 

В этом файле представлена сводка содержимого всех файлов, входящих в 
состав приложения Project3.

Project3.vcproj
   Основной файл проекта VC++, автоматически создаваемый с помощью 
   мастера приложений. 
   В файле представлены сведения о версии Visual C++, используемой для 
   создания файла, а также сведения о параметрах платформы, конфигурации 
   и проекта, установленных с помощью мастера приложений.

Project3.cpp
   Основной исходный файл приложения.
   Код для отображения формы.

Form1.h
   Реализация класса формы и функции InitializeComponent().

AssemblyInfo.cpp
   Пользовательские атрибуты для изменения метаданных сборки.

/////////////////////////////////////////////////////////////////////////////

StdAfx.h, StdAfx.cpp

   (PCH) Project3.pch и файла предкомпилируемых типов 
   StdAfx.obj.
